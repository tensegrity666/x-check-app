#### Clone repo
`$ git clone git@github.com:tensegrity666/x-check-app.git`<br>
<br>
_then checkout to_ __origin/dev__:<br>
`$ git fetch origin`<br>
`$ git checkout -b dev origin/dev`

#### Install
`$ npm i`

#### Start server
`$ npm start`<br>
_Open_ [http://localhost:3000](http://localhost:3000) _to view it in the browser._

#### Build project
`$ npm run build`
