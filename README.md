#### Clone repo
`git clone git@github.com:tensegrity666/x-check-app.git`<br>
then checkout to __origin/dev__:<br>
`git fetch origin`
`git checkout -b dev origin/dev`

#### Install
`npm i`

#### Start server
`npm start`

#### Build project
`npm run build`
